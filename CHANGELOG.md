## [Unreleased]

### Added
- Initial project structure
- Farcaster Mini App integration
- Next.js 15 with App Router
- shadcn/ui component library
- Tailwind CSS styling
- Nix-based build system

## [1.0.0] - 2024-XX-XX

### Added
- Daily fantasy leagues functionality
- Crypto integration with USDC
- Live leaderboards
- Multi-asset drafting (memecoins, traders, athletes)
- Cross-platform support (iOS, Android, Web)
- Modern UI with shadcn/ui and Tailwind CSS
- Farcaster SDK integration
- Xnode deployment configuration

### Changed
- Project structure refactored for better organization
- Documentation improved and expanded
- Code quality improvements

### Security
- Environment variable configuration
- Secure account association setup
